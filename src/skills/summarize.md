Summarize the available transcript context.

Requirements:

- Start with a 2-4 sentence overview.
- Then list the most important points as concise bullets.
- Cite transcript timestamps for each important point.
- Prefer the user's selected time window if one is active.
- Keep the answer compact unless the user asks for detail.
